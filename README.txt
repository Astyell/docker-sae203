QUEL EST CE POKEMON ?
Le site est un quizz pour trouver le Pokemon générer aléatoirement.
Vous avez le droit à plusieurs essaies pour deviner le Pokémon adéquoit. Le joueur à accès à l'historique de ces anciennes suppositions qu'il lui sert d'indices : si une des caractéristiques du Pokémon à trouver est similaire avec le Pokémon deviné.

Si le joueur à besoin d'aide pour trouver le Pokémon, il a accès  à un Pokédex qui recensie tous les Pokémons qui peuvent être deviner.
