QUEL EST CE POKEMON ? 
Un quizz pour trouver un Pokemon générer aléatoirement !

Vous avez le droit à plusieurs essaies pour deviner le Pokémon adéquoit.
Le joueur à accès à l'historique de ces anciennes suppositions qu'il lui sert d'indices :
    si une des caractéristiques du Pokémon à trouver est similaire avec le Pokémon deviné.

Pour trouver le Pokémon, vous avez accès à un POKEDEX qui recensie tous les Pokémons qui peuvent être deviner.

Nous avons laisser le nom du Pokémon à trouver en haut à gauche
pour le confort de nos professeurs.
